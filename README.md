# Beginner-Friendly Java Calculator

This project contains a simple **Java Calculator** designed to help new
programmers practice:

-   Variables\
-   Operators\
-   Decision-making (if--else)\
-   User input (Scanner)

The file **Calculator.java** includes blanks (`// TODO`) where you must
insert missing lines of code before running the program.

## ‚ñ∂Ô∏è How to Run the Program

Follow these steps to compile and run the Java calculator.

------------------------------------------------------------------------

## 1. Install Java

Make sure Java is installed.

Check version:

``` sh
java -version
```

If Java is not installed, watch our video on installing Java: https://sign-java.netlify.app/gettingstarted/install_java.html

------------------------------------------------------------------------

## 2. Save the File

Save the Java source code as:

    Calculator.java

Make sure the file name **matches the class name** exactly.

------------------------------------------------------------------------

## 3. Open Terminal / Command Prompt

Navigate to the folder where your file is saved.

### Windows example:

``` sh
cd C:\Users\YourName\Desktop
```

### Mac/Linux example:

``` sh
cd /Users/yourname/Desktop
```

------------------------------------------------------------------------

## 4. Compile the Code

Run this command:

``` sh
javac Calculator.java
```

If there are no errors, this will generate:

    Calculator.class

------------------------------------------------------------------------

## 5. Run the Program

Execute the calculator:

``` sh
java Calculator
```
